# Sugg — Discord Suggestion Bot

**Developer:** Saif Fikry

بوت ديسكورد عربي متكامل لإرسال الاقتراحات كبطاقات صور احترافية مع أزرار تصويت.

---

## المميزات (Features)

| الميزة | الوصف |
|---|---|
| **بطاقات صور احترافية** | كل اقتراح يتحول لبطاقة تصميم فريدة بـ Canvas بأبعاد 1000×560 |
| **5 ثيمات ألوان مخصصة** | ثيمات مختلفة للبطاقة بناءً على المحتوى: Default, Ocean, Sunset, Nature, Modern |
| **خط عربي كامل** | دعم كامل للغة العربية (RTL) بخط Tahoma مع fallback للايموجي |
| **تكبير الخط تلقائياً** | النص الطويل يتصغر تلقائياً (auto-fit) عشان يفضل داخل البطاقة |
| **شريط تقدم مرئي** | يظهر تفاصيل التصويت (نعم / لا) مع شريط تقدم ونسب مئوية |
| **أزرار تصويت** | كل بطاقة فيها زراين ✅ و ❌ — التصويت فوري بدون إنشاء ثريد |
| **تحديث مباشر** | عند التصويت، البطاقة تتجدد فوراً بالنتائج الجديدة |
| **Badge الرتبة** | تظهر رتبة صاحب الاقتراح بجانب اسمه على البطاقة |
| **علامة مائية** | كل بطاقة عليها علامة "sv1ffff" لحماية المحتوى |
| **بدون ثريد** | البوت ما يفتح ثريد — الرد يكون بصورة + أزرار فقط |
| **تصويت النفس مسموح** | صاحب الاقتراح يقدر يصوت على اقتراحه |
| **كود مشفر** | الكود بالكامل مشفر (Obfuscated) لمنع القراءة أو التعديل |

## طريقة التشغيل

### 1. متطلبات (Requirements)
- Node.js v20+
- npm

### 2. التثبيت
```bash
npm install
```

### 3. الإعداد (Configuration)
أنشئ ملف `.env` في مجلد المشروع:

```
TOKEN=هنا_توكن_البوت
SUGGESTION_CHANNEL_ID=ايدي_روم_الاقتراحات
```

### 4. التشغيل

**للمستخدم (النسخة المشفرة):**
```bash
npm start
```

**للمطور (النسخة الأصلية من الباكاب):**
```bash
npm run dev
```

**إعادة التشفير بعد التعديل:**
```bash
npm run build
```
